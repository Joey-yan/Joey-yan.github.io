利用github-pages建立个人博客

发表于 2016-07-31   |  

引言

之前就一直想搭建一个属于自己的博客，记录自己所学所想。通过知乎了解到：利用github-pages可以搭建一个免费的个人博客，于是我马上动起手来。

第一次搭建博客失败，当时是使用Jekyll搭建的，苦于没有找到好的主题，自己动手配置起来所需要的时间成本太大，所以放弃了。然后又是逛知乎，知道了除Jekyll 之外，还可以用Hexo来搭建博客。ps: 当时就是被Next主题吸引了。

本地搭建博客

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown （或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

安装前提

安装Hexo之前，必须确定电脑中已安装下列应用程序：

  • Node.js
  • Git or GitHub Desktop

部署到GitHub

静态化处理

要说明的一点是Hexo的静态博客框架。静态博客：是只包含html、css、javascript文件的网站，没有动态的脚本。虽然我们用Node进行开发，但博客的发布后就与Node无关了。在发布之前，我们要通过一条命令，把所有的文章都做静态化处理，就是生成对应的html、css、javascript，使得所有的文章都是由静态文件组成的。

在执行hexo deploy 命令时，可能会提示找不到git 解决方法；在Hexo 3.0版本后deploy git被分开的，所以需要安装，安装命令如下： npm install hexo-deployer-git --save , 安装好后再尝试一下就好了
