Sublime Text3 使用说明书

发表于 2016-08-15   |  

这篇文章主要记录使用 Sublime Text3 过程中点滴，我会不定时的更新。
如果能对其他朋友有所帮助，也是极好的。

前言

当初选择编辑器主要是参考我最喜欢的软件这个网站，因为 Notepad++ 排在第一位，也就很自然的选择了 Notepad++（ps:除了颜值低了点，其他都很不错）。也正是这次机缘，让我邂逅了 Sublime Text 这款文本编辑器。

Sublime Text 官网上还是以 Sublime Text2 为例。我个人认为 Sublime Text3 的界面更很好看一些，如果你是 Win10 系统。所以，没下手的赶紧安装吧。

安装

Sublime Text is available for OS X, Windows and Linux.

Sublime Text3 是一个跨平台的文本编辑器，支持 OS X， Windows 和 Linux。进入 官网 选择对应的平台，安装即可。我自己使用的是 Windows 系统，下面所述都是基于 Windows 的，其他平台不作保证。

没错，本篇博文就是在 Sublime Text3 上写的。前几篇博文都是用 Typora 编写的，这是一款实时预览的 Markdown 编辑器。

直接安装

安装 Sublime Text3 插件很方便，可以直接下载安装包解压到 Packages （Preferences->Browse Packages） 目录。

个性化设置

更换字体

不管你是否喜欢默认的字体，Sublime Text3 都是可以更换字体的，方式很特别，你得通过配置文件进行设置。
点击进入 Settings-User （Preferences->Settings-User） ，添加 font_face 属性：

{
“font_face”: ‘YaHei Consolas Hybrid’,
“font_size”: 10
}

关于 YaHei Consolas Hybrid 的介绍，应用《程序编写字体 — Yahei Consolas Hybrid》中的一段话：

Yahei Consolas Hybrid 是一种合成字体，是由微软雅黑的中文字体和 Consolas 的英文字体合成的，Consolas
字体是微软专为程序员开发的一种字体，但没有中文支持，所以 Yahei Consolas Hybrid 这种字体特别适合中英文的编程开发，代码编写。
